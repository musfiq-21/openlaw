# 🏛️ ConstitutionBD - সম্পূর্ণ ব্যবহারকারী গাইড

## 📖 কী এটি?

**ConstitutionBD** একটি উন্নত AI সিস্টেম যা বাংলাদেশ সংবিধান সম্পর্কে জটিল প্রশ্নের উত্তর দেয় **সাধারণ বাংলা ভাষায়**। এটি রাগ (RAG) প্রযুক্তি এবং গুগল জেমিনি ব্যবহার করে।

## 🚀 দ্রুত শুরু (৫ মিনিটে)

### যদি আপনি Windows ব্যবহার করেন:
```bash
startup.bat
```

### যদি আপনি macOS/Linux ব্যবহার করেন:
```bash
chmod +x startup.sh
./startup.sh
```

তারপর মেনু থেকে **অপশন 1** নির্বাচন করুন।

## 🎯 প্রথম পদক্ষেপ

### 1️⃣ Google API কী পান (বিনামূল্যে)
```
https://makersuite.google.com/app/apikey
```

### 2️⃣ .env ফাইল তৈরি করুন
```bash
cd backend
copy .env.example .env  # Windows
# অথবা
cp .env.example .env    # macOS/Linux
```

### 3️⃣ API কী যোগ করুন
.env ফাইল খুলুন এবং যোগ করুন:
```
GOOGLE_API_KEY=আপনার-API-কী-এখানে
```

### 4️⃣ সবকিছু চালু করুন
```bash
startup.bat          # Windows
# অথবা
./startup.sh         # macOS/Linux
```

### 5️⃣ ব্রাউজারে খুলুন
```
http://localhost:3000
```

## 📱 ইউজার ইন্টারফেস ব্যবহার করুন

### প্রশ্ন করার ট্যাব 🔍
```
1. প্রশ্ন লিখুন (উদাহরণ: "মৌলিক অধিকার কি?")
2. "প্রশ্ন জমা দিন" ক্লিক করুন
3. সিস্টেম ২-৫ সেকেন্ডে উত্তর দেবে
4. উৎস এবং উদ্ধৃতি পড়ুন
```

### সার্চ করার ট্যাব 🔎
```
1. বিষয় লিখুন
2. সিমান্টিক সার্চ চালু হবে
3. সম্পর্কিত অনুচ্ছেদ দেখুন
```

### ইতিহাস ট্যাব ⏱️
```
1. আপনার সাম্প্রতিক প্রশ্নগুলি দেখুন
2. যেকোনো প্রশ্ন পুনরায় চালাতে পারেন
```

## 🐛 সমস্যা সমাধান

### সমস্যা: "Connection refused"
**সমাধান:**
```bash
# নতুন টার্মিনালে ব্যাকএন্ড চালু করুন
python main.py
```

### সমস্যা: "ModuleNotFoundError"
**সমাধান:**
```bash
pip install -r backend/requirements.txt
```

### সমস্যা: ফ্রন্টএন্ড খালি দেখাচ্ছে
**সমাধান:**
1. F12 চাপুন (ডেভেলপার কনসোল)
2. কোনো লাল ত্রুটি দেখুন?
3. নেটওয়ার্ক ট্যাব খুলুন
4. /health কল সাফল্যজনক কিনা দেখুন

### সমস্যা: "CORS Error"
**সমাধান:**
- নিশ্চিত করুন ব্যাকএন্ড `http://localhost:8000` এ চলছে
- ফ্রন্টএন্ড। `http://localhost:3000` এ চলছে

## 🔧 API এন্ডপয়েন্ট

### প্রশ্ন জিজ্ঞাসা করুন
```bash
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "মৌলিক অধিকার কী?",
    "include_sources": true
  }'
```

### সার্চ করুন
```bash
curl -X POST "http://localhost:8000/search" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "নাগরিক অধিকার",
    "k": 5
  }'
```

### স্বাস্থ্য পরীক্ষা করুন
```bash
curl http://localhost:8000/health
```

জনপ্রিয় প্রশ্নগুলির উপর ক্লিক করুন - সেগুলি স্বয়ংক্রিয়ভাবে পূরণ হবে।

### ইতিহাস পুনরায় চালান
পূর্ববর্তী প্রশ্নগুলি পুনরায় চালানোর জন্য "পুনরায় চালান" বোতাম ব্যবহার করুন।
